## 📌 Project Overview

The **Smart Attendance System** is a web-based application developed using PHP and MySQL.
It automates attendance using **QR codes with time validation**, reducing manual effort and preventing proxy attendance.

---

## 🎯 Objectives

* ✔ Automate attendance system
* ✔ Reduce manual effort
* ✔ Prevent proxy attendance
* ✔ Maintain digital records
* ✔ Generate attendance reports

## 🔥 Key Features

* 📱 QR Code based attendance
* ⏱ Time validation (Present / Late / Absent)
* 👨‍🏫 Teacher Dashboard
* 👨‍🎓 Student Dashboard
* 📊 Attendance Reports with Charts
* 📥 Export attendance to Excel
* 🔐 Secure login system

---

## 🧩 System Modules

### 👨‍🏫 Teacher Module

* Create lecture
* Generate QR code
* View student attendance
* Monitor lectures

### 👨‍🎓 Student Module

* Login/Register
* Scan QR code
* Mark attendance
* View reports

---

## 🔄 Workflow

1. Teacher logs in
2. Creates lecture
3. QR code generated
4. Student scans QR
5. Attendance recorded
6. System assigns status:

   * Present
   * Late
   * Absent
7. Reports generated
